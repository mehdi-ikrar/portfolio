# Portfolio Developer Pro

## Description
Ce projet est un portfolio en ligne pour un développeur web professionnel. Il présente les compétences, les projets réalisés, et permet aux visiteurs de le contacter facilement.

## Structure du projet
Le portfolio est organisé de la manière suivante :

- **index.html** : Page d'accueil du portfolio.
- **assets/** : Dossier contenant les ressources du projet.
  - **css/** : Dossier pour les fichiers CSS.
    - **style.css** : Styles principaux du portfolio.
    - **responsive.css** : Styles pour la réactivité du site.
  - **js/** : Dossier pour les fichiers JavaScript.
    - **main.js** : Code JavaScript principal pour les interactions.
  - **fonts/** : Dossier pour les polices personnalisées.
    - **fonts.css** : Gestion des polices utilisées dans le portfolio.
- **projects/** : Dossier contenant la page dédiée aux projets.
  - **index.html** : Présentation des projets réalisés.
- **about/** : Dossier contenant la page "À propos".
  - **index.html** : Informations sur le développeur.
- **contact/** : Dossier contenant la page de contact.
  - **index.html** : Formulaire de contact.
- **resume/** : Dossier contenant la page de CV.
  - **index.html** : Présentation du parcours professionnel.

## Installation
1. Clonez le dépôt sur votre machine locale.
2. Ouvrez le fichier `index.html` dans votre navigateur pour voir le portfolio.

## Déploiement
Pour déployer le portfolio, vous pouvez utiliser des services d'hébergement comme GitHub Pages, Netlify ou Vercel. Suivez les instructions spécifiques à chaque service pour mettre en ligne votre portfolio.
